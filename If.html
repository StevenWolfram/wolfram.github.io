Услоный оператор If

If - самый простой условный оператор

Стуркутра If:
If[условие, оператор true, оператор false, оператор default]

Пример:
In[1]: If[3>1, "Выполняется", "Не выполняется", "Не понял"]
Out[1]: Выполняется

In[2]: If[0>1, "Выполняется", "Не выполняется", "Не понял"]
Out[2]: Не выполняется

In[3]: If[{1,2,3}>1, "Выполняется", "Не выполняется", "Не понял"]
Out[3]: Не понял
